Instructions

SETUP: Cluster all Blinks into the starting arrangement. Long press changes a Blink’s team color. Each Player chooses a color and orients the board so that their color faces them. Double-click any Blink to start the game.

GAMEPLAY: Each Blink, or “Mortal”, starts with 60 seconds of life. The only way to extend a Mortal’s life is to steal seconds from others. Players take turns removing an accessible Mortal from the cluster and placing it in a new location. A single Mortal that is removed from the cluster goes into ‘attack’ mode; when it rejoins the cluster, it steals 5 seconds from each surrounding Mortal, including from its own team.

The green player removes a Blink and makes an attack

Not all Mortals can be legally removed from the cluster at the start of a turn. Mortals with neighbors on opposing faces are “locked,” as are Mortals that, if removed, would fracture the cluster into more than 2 pieces.
