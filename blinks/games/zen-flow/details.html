Instructions

GAMEPLAY: SPREADING A SINGLE COLOR Single-click spreads the next color in a wave across all connected Blinks.

Single color wave spreading across a field

SPREADING RANDOM COLORS Double-click sparkles random colors in a wave across all connected Blinks.

Random color wave spreading across a field

MIRROR MODE Long press changes a Blink from color-spreading mode to mirror mode, which displays the colors of the Blinks it is connected to.

Mirror mode Blinks displaying neighboring colors

Win Conditions

Zen Flow was designed as a meditative experience, so there is no implicit goal. Switch Blinks between modes and create new arrangements to explore this Zen space.
